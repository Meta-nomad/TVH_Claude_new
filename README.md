# TradingView History Bot v2

Находит биржу с **максимальной историей цены** для любой криптовалюты.
При одинаковой глубине истории выбирает биржу с **наименьшим количеством разрывов** в часовом графике.

---

## Установка

```bash
# 1. Распакуйте архив и войдите в папку
cd tgbot2

# 2. Создайте виртуальное окружение
python3 -m venv venv
source venv/bin/activate          # Linux / macOS
# venv\Scripts\activate           # Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Создайте .env
cp .env.example .env
# Откройте .env и вставьте токен бота от @BotFather

# 5. Запустите
python main.py
```

## Источники данных (в порядке приоритета)

| Источник | Что даёт | Ключ нужен? |
|---|---|---|
| CryptoCompare | Биржи, история, hourly gap | Нет (с ключом больше лимит) |
| Binance | Подтверждение пар, история, hourly gap | Нет |
| Kraken | Подтверждение пар, история | Нет |
| CoinGecko | Genesis date, дополнительные биржи | Нет (Pro = меньше ограничений) |

Если один источник недоступен — бот автоматически переключается на следующий.

---

## Логика выбора

1. **Монета найдена** — проверяется через CryptoCompare / Binance / Kraken / CoinGecko
2. **Дата запуска** определяет приоритет котировок:
   - Старше USDT (до 2014-10-06): `USD → BTC → ETH → USDT`
   - Новее USDT: `USDT → USD → BTC → ETH`
3. **Дата начала истории** — основной критерий выбора биржи
4. **Gap score** — при одинаковой дате (±3 дня) побеждает биржа с меньшим количеством разрывов в часовом графике

## Опциональный ключ CryptoCompare

Без ключа: 100 000 запросов/месяц.
С бесплатным ключом: 250 000 запросов/месяц.

Получить: https://www.cryptocompare.com/cryptopian/api-keys  
Вставить в `.env`:
```
CRYPTOCOMPARE_API_KEY=ваш_ключ
```
